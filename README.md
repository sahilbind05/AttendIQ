# 📊 AttendIQ — Student Attendance Tracker

A modern, smart attendance tracker for students.

## 🚀 Live Demo
👉 [Click here to try it](https://sahilbind05.github.io/AttendIQ/)

## ✨ Features
- Track attendance for multiple subjects
- Smart calculation — tells you classes needed to reach 75%
- Dark/Light mode
- Data saved automatically in your browser
- No login, no installation needed

## 🛠️ Built With
- HTML, CSS, JavaScript
- Runs entirely in the browser
